## 这是什么东西

这是一款chrome浏览器插件

## 为什么会有他

都说世界的进步是被懒人驱动的。这是我生活中的一个强需求，经常需要把PC端的网页发到手机上查看。而以前繁琐的操作令我深恶痛绝~~~ 于是我决定一懒到底

## 他能干什么

他能快速的把一个打开的网页发送到手机并在手机上浏览

## 他的应用场景

1. 浏览一个网页想看看手机上的效果
2. 新闻看了一半，下班了，发手机路上看吧
